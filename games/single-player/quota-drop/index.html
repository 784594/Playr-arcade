    How It Plays

    • 1. Each pull spends one round from the chamber timer.
    • 2. Cash persists between levels, so a strong streak can front-load later quotas.
    • 3. Meeting the quota at or before zero rounds lifts you into the next level and refreshes the timer.
    • 4. Missing the quota when rounds run out triggers the trap door and ends the run.

    Probability Tuning

    The slot math lives in two places inside the script: the reel strips and the payout table. Add more copies of a symbol to make it more common, or change a payout value to tune how lucrative it is.

    const REEL_STRIPS = [ ['gear', 'bar', 'ember', 'gear', 'skull', 'seven', 'bar', 'vault', 'gear', 'ember', 'wild', 'gear'], ... ]; const PAYOUT_RULES = { seven: { triple: 140, pair: 28, singleBonus: 0 }, wild: { triple: 220, pair: 44, singleBonus: 0 }, ... };
